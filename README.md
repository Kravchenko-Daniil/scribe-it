## 🌱 Что это

Если тебе надо получить текст интервью, подкаста, лекции или совещания — обычно приходится либо расшифровывать руками, либо идти на платные сервисы с веб-интерфейсом и аккаунтами. Это долго.

**scribe-it — это ярлык в Telegram, который делает то же самое за минуту.** Кидаешь боту файл или ссылку → получаешь назад текст с абзацами и разделением по спикерам, субтитры с таймкодами и сырой JSON от [ElevenLabs Scribe](https://elevenlabs.io/speech-to-text). Никаких регистраций и веб-интерфейсов.

## ⚡ Как пользоваться

Открываешь бота → отправляешь **файл** или **ссылку**:

| Что принимает | Лимит |
|---|---|
| Видео / аудио / voice / document в Telegram | до 20 МБ (ограничение Telegram Bot API) |
| Ссылка на YouTube | любая длина |
| Прямая ссылка (Яндекс.Диск, Google Drive, http) | любой размер |

В ответ приходит **три файла:**
- `*.txt` — чистый текст с абзацами и разделением по спикерам
- `*.srt` — субтитры с таймкодами на уровне слов (готовы для видеоредактора)
- `*.json` — сырой ответ Scribe на случай, если нужно что-то своё строить поверх

Большие видео больше 20 МБ проще залить на YouTube как unlisted или на Яндекс.Диск и прислать ссылку — бот сам скачает.

## 🔧 Под капотом

```
Telegram → bot.py (aiogram)
             │
             ├── файл: напрямую скачивает через Bot API
             └── ссылка: yt-dlp извлекает аудио
                   └── ffmpeg пережимает в Opus 32 kbps
                         └── scribe.py → ElevenLabs Scribe API
                               └── отправка .txt / .srt / .json обратно
```

- **`bot.py`** — aiogram-приложение, фильтрация разрешённых пользователей, единая очередь заданий
- **`downloader.py`** — yt-dlp в двух режимах (YouTube / универсальный) + ffmpeg pipeline
- **`scribe.py`** — клиент ElevenLabs Scribe, конвертер в .srt на уровне слов
- **`storage.py`** — per-request tmp-директории с авто-очисткой старых
- Запускается как systemd-сервис, зависимости через `uv`

## 📦 Запуск

```bash
cp .env.example .env      # заполнить TELEGRAM_BOT_TOKEN, ELEVENLABS_API_KEY, ALLOWED_USER_IDS
uv sync
uv run python bot.py
```
